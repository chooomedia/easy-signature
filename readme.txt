Ein Plugin zur Erstellung anpassbarer Mitarbeitersignaturen für WordPress.

== Description ==

Easy Signature ermöglicht es Ihnen, professionelle E-Mail-Signaturen für Ihre Mitarbeiter zu erstellen und zu verwalten. Mit anpassbaren Feldern und Designoptionen können Sie Signaturen erstellen, die perfekt zu Ihrem Unternehmen passen.

Hauptfunktionen:
* Erstellung individueller Signaturen für jeden Mitarbeiter
* Anpassbare Felder für Kontaktinformationen und Social-Media-Links
* Einfache Integration von Profilbildern und Unternehmenslogos
* Responsive Design für optimale Darstellung auf allen Geräten

== Installation ==

1. Laden Sie das Plugin-Verzeichnis in das `/wp-content/plugins/`-Verzeichnis hoch
2. Aktivieren Sie das Plugin über das 'Plugins'-Menü in WordPress
3. Gehen Sie zu "Signaturen" im WordPress-Admin-Bereich, um neue Signaturen zu erstellen

== Frequently Asked Questions ==

= Kann ich mehrere Signaturen für einen Benutzer erstellen? =

Ja, Sie können beliebig viele Signaturen für jeden Benutzer erstellen.

= Ist das Plugin DSGVO-konform? =

Ja, das Plugin speichert alle Daten lokal in Ihrer WordPress-Installation und teilt keine Informationen mit externen Diensten.
